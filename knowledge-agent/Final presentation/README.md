# Frantelligence Final Slide Deck

Built with [Reveal.js 4.6](https://revealjs.com/) and CDN assets.

## Files
- `index.html` - runnable final deck scaffold and styling
- `slides.md` - plain-text outline for slide planning

## Open the deck
- Open `index.html` with Live Server in VS Code, or open directly in a browser.

## Theme notes
- Uses a dark palette with high contrast and cyan accents inspired by the IS 590R course site aesthetic.
- Update theme variables in the `:root` block in `index.html`.
